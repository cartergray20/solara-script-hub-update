## FAQ

**Is Solara PC safe to use?**  
Running third-party executables together with Roblox is the user's responsibility. Check the license and Roblox terms before you proceed.

**Will this work with future Roblox updates?**  
The auto-update engine is intended to deliver compatibility patches, but ongoing support is not guaranteed.

**How does Solara PC compare to other script executors?**  
Its main emphasis is a lightweight footprint and batch execution, which can set it apart from other tools in terms of behavior and available features.

**Can my Roblox account be affected?**  
External execution tools may conflict with Roblox terms of service. Use it with awareness of possible account consequences.

**Where are my custom scripts stored?**  
Custom user scripts live in the local `scripts/user/` directory inside the application folder, so they remain available between sessions.
